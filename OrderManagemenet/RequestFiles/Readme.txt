Application ASP.NET EntityFramework Local(Db)

1. You can add new data in the database from files with extension .json, .csv, .xml.
Choose file on the your computer and Click on the Button "Add Data". File will be saved on Server, information added in database.

2. After adding data you can watch information on the screen;

3. You can sort columns by increasing and decreasing. 
Click on the Title of Column (RequestId, ClientId, NameQuantity, Price) and information will be sorted.

4. You can get report as .json file. Click on Button "Create Report" and you will see reference
for download report. It will create new file on Server which you can download using reference "Download Report".
